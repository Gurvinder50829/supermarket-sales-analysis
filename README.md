📊 Superstore Dataset Analysis Project
📌 1. Overview

This project performs an exploratory data analysis (EDA) on the Superstore dataset to understand business performance across sales, profit, customer segments, categories, and regions.
The main goal is to generate data-driven insights for improving business decisions.

🎯 2. Problem Statement

Retail businesses often face challenges such as:

Identifying low-profit products
Understanding customer buying patterns
Improving regional performance
Optimizing sales strategy

This project aims to solve these using data analysis.

📂 3. Dataset Information

The dataset contains retail transaction records including:

Order ID & Order Date
Product Category & Sub-category
Sales & Profit
Customer Segment
Region & State
Shipping Mode
